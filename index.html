Reason Foundation GitHub Page

This website will cover R packages for pension analysis, and other coding done by Reason Foundation.

  • "pensionviewr" package allows to pull/filter/visualize data from pension database in R.

    • Package:

    https://github.com/ReasonFoundation/pensionviewr

    DataBase Repo:

    https://github.com/ReasonFoundation/databaseR
  • "reasontheme" package allows to set Reason theme for ggplo2 visualizations.

    • Package:

    https://github.com/ReasonFoundation/reasontheme
  • GraphicsR repo contains current work on creating graphs for & automating pension decks in R .
    • https://github.com/ReasonFoundation/GraphicsR
